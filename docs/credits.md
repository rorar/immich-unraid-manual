## Immich Team

[Immich](https://immich.app/) is an incredible open-source project for self-hosted photo and video management. If you find Immich useful, consider supporting the project by donating or contributing to the codebase.

## PhotoMigrator Team

[PhotoMigrator](https://github.com/jaimetur/PhotoMigrator/) simplifies migration from Google Photos and other services to Immich — handling metadata, folder structure, and album creation. If you find PhotoMigrator useful, consider supporting the project by donating or contributing to the codebase.
